Uopšteno

Ovdje se radi o projektu, koji sam dobio sasvim slučajno od jednog priatelja koji se javio nekim amerikancima developers at x-team na konkurs i oni mu poslali ovo. Svidio mi se iz vise razloga. Radilo se o klasicnom backend zadatku (u prvom mahu), a osim toga dosta se razlikova od svih mojih predhodnih pokusaja pisanja koda - klasicni upis i ispis podataka iz baze uz pomoc Spring boot-a i hibernate, pa njihovo reprezentovanje uz pomoć JSP-a (Spring MVC) git link .
Ovdje sam kao pozadinu koristio RESTful Web Service,a projekat konfigurisao uz pomoc Spring boot-a, i to je bilo to...dok nakon nekog vremena nisam uvidio (uz pomoć prijatelja naravno, opet) da se mora nešto poraditi i po pitanju fronta. Jedan od mojih prvih dodira sa front tehnologijama sam pokušao iskoristiti na nadogradnji ovog projekta. Tako da sam uspio nešto odraditi sa Angularom, ne JS-om, već sa verzijom 2 +...!

    Sto se tiče razvojnog okruzenja (IDE) koje sam koristio:
  • STS (Spring Tool Suite). U stvari radi se o klasičnoj varijanti eclipse, sa poboljšanjima vezanim za Spring pre svega,
  • Visual Studio Code sam upražnjavao kod Angulara.
